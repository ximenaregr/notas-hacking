## Descripción

This is a really weird text file [TXT](https://jupiter.challenges.picoctf.org/static/e7e5d188621ee705ceeb0452525412ef/flag.txt)? Can you find the flag?
## Solución
Descargué el archivo y lo abrí vemos que en la primera línea aparece PNG, así que quizás sea solo una imagen. Para abrir el archivo como imagen, necesitamos cambiarle la extensión. Ahora sí podemos abrirlo como imagen. Y ahí esta la bandea.
