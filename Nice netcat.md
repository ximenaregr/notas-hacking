## Descripción

There is a nice program that you can talk to by using this command in a shell: `$ nc mercury.picoctf.net 43239`, but it doesn't speak English...
## Solución

En este ejercicio use el webshell de picoCTF, usando una linea de comando de 'nc mercury.picoctf.net 43239', para abrir la coneccion con el servidor 43239, despues nos dio una lista de numeros a la cual le aplique la formula de From decimal usando cyber chef.

La respuesta fue:picoCTF{g00d_k1tty!_n1c3_k1tty!_7c0821f5}
