## Descripción

Run the Python script and convert the given number from decimal to binary to get the flag.[Download Python script](https://artifacts.picoctf.net/c/22/convertme.py)
## Solución

Descargamos el archivo, al correr el script nos pide que concirtames un numero de decimal a binario para regresarnos la bandera.

```
ximenare-picoctf@webshell:~$ python convertme.py
If 90 is in decimal base, what is it in binary base?
Answer: 1011010
That is correct! Here's your flag: picoCTF{4ll_y0ur_b4535_762f748e}
```
